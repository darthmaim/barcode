# `@darthmaim/react-native-barcode`

React-Native Component to render EAN barcode as svg.

## Usage

```
import { Barcode } from '@darthmaim/react-native-barcode';

<Barcode value="4003994155486"/>
```

## Installation

```
npm i @darthmaim/react-native-barcode
```
