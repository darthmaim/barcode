# `@darthmaim/barcode`

Library to validate and format EAN barcodes.

## Usage

```
import { isValid } from '@darthmaim/barcode/lib/ean13';

isValid("4003994155486");
```

## Installation

```
npm i @darthmaim/barcode
```
