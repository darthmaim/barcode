# `@darthmaim/react-barcode`

React Component to render EAN barcode as svg.

## Usage

```
import { Barcode } from '@darthmaim/react-barcode';

<Barcode value="4003994155486"/>
```

## Installation

```
npm i @darthmaim/react-barcode
```
